Welcome to WebPulse

As a self-taught developer without formal work experience in the tech industry, I created a full-stack app that serves as a central repository for development resources. This app enables users to register, log in, reset passwords, and manage, read, and publish blog posts.

It serves as a proof of my skills in software development, including the planning, development, and deployment of complex web applications.

https://webpulseui.web.app

Project Details

Technologies & Tools

  • Frontend: Angular 16, Bootstrap, Angular CLI, Responsive CSS, JWT, RxJs
  • Backend: ASP.NET Core 7, SQL Database, Azure, REST-APIs
  • Deployment: Azure (Backend), Firebase (Frontend), GitHub Actions
  • Version Control: Git, Visual Studio Code, Visual Studio

Main Features

  • RBAC: Role-based system with Watcher, Reader, and Writer access.
  • Content Management: Writers can manage and publish posts.
  • Security: Authentication and Authorization with JWT.
  • Responsive design: optimized for multiple devices.

Challenges & Learnings

  • Self-learning through tutorials, documentation, and practical application.
  • Adapting design for various device resolutions.
  • Implementing complex features like role-based access.
